Iteration

Omelet allows you to iterate over variables in your context with for...in loops. This is most useful when you have an array in your context, and you want to display its contents in your template somehow. You can iterate over it, printing a block of template code once per element in the array.

Note that since Omelet (by design) never raises type errors, if you try to iterate over an object that is not an array, it will be treated as if it were an array of length one, with itself as its only element. In the example below, we try to iterate over the number 47. The loop is executed once, with the iterator set to 47 for that single iteration.

For these examples, assume we have in our context an array called people with the value ["Alice","Bob","Carol"], and a number called number with the value 47.

Omelet template:
>for person in people:
    The current person is {person}.

>for n in number:
    The current number is {n}.
Rendered HTML:
The current person is Alice.
The current person is Bob.
The current person is Carol.

The current number is 47.
