Conditionals

Omelet allows for simple conditionals in the form of if, elif, and else statements. A conditional statement can have zero or more elif branches, and zero or one else branches. These look much like they do in other languages, but Omelet has one major restriction: the predicate of if and elif statements can only be the name of a variable.

A predicate is considered false if the variable referenced is any of the following: a boolean with the value false, null, or undefined (using JavaScript's definition of undefined). The predicate holds in all other cases.

Complex boolean expressions are not allowed in Omelet, but you can negate the predicate variable by prepending it with a ! character. In this case, the "then" case of the conditional statement gets executed when the predicate is considered false.

In the examples below, assume we have in our context a variable b with the value true, a variable n with the value 47, and no defined variable called unDef.

Omelet template:
>if b:
    This will be printed.
>else:
    This will not be printed.

>if !n:
    This also will not be printed.
>elif !unDef:
    This will also be printed.
Rendered HTML:
This will be printed.
This will also be printed.
Omelet is a template language developed by Reid Mitchell.
