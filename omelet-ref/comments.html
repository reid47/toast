Comments

Omelet supports single-line comments preceded by ##. When the parser encounters two consecutive # characters, it considers the rest of the line a comment.

Here is some sample Omelet code containing comments, followed by the HTML that it is rendered into.

Omelet template:
This is one line
This is another line ## with a partial-line comment at the end
## This is a third line that is fully commented-out
Rendered HTML:
This is one line
This is another line

If you want to include two consecutive # characters in a template and have them not be recognized as the beginning of a comment, escape the second one with a \ character, as shown below.

Omelet template:
This text contains the characters #\#, escaped.
Rendered HTML:
This text contains the characters ##, escaped.
