Includes

Omelet allows you to include the contents of other template files in your templates. This is most useful for splitting templates into small, reusable pieces.

To include the contents of another template, you use the expression {>include fileName.oml}. This is evaluated exactly the same as if you had copy/pasted the contents of fileName.oml into your current template file. The included file is evaluated within the context of the template that is including it.

Let's call this file sidebar.oml:

Omelet template:
## sidebar.oml

@ul.side-bar
    @li|a[href="link1.html"] Navigation link 1
    @li|a[href="link2.html"] Navigation link 2
    @li|a[href="link3.html"] Navigation link 3

Now, assuming we are writing a template that will be located in the same directory as sidebar.oml (so that we do not need to worry about a complex relative path name), we can drop the contents of sidebar.oml into our template like this:

Omelet template:
@html
    @head
        @title My website
    @body
        {>include sidebar.oml}
        @div.contents
            These are some other contents of the page!
Rendered HTML:
<html>
    <head>
        <title>My website</title>
    </head>
    <body>
        <ul class="side-bar">
            <li><a href="link1.html">Navigation link 1</a></li>
            <li><a href="link2.html">Navigation link 2</a></li>
            <li><a href="link3.html">Navigation link 3</a></li>
        </ul>
        <div class="contents">
            These are some other contents of the page!
        </div>
    </body>
</html>
